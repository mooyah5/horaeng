## 💡 서비스 소개

### 에너지 절약 모바일 애플리케이션

> 매년 늘어나는 멸종위기 동물들 <br />
> 에너지 절약을 통해 탄소배출을 줄인다면 이런 동물들이 더 잘 살아갈 수 있지 않을까? <br />
> 소소한 일상 속의 에너지 절약을 통해 멸종위기동물을 구해보세요!

#### 매일 꾸준한 에너지 절약을 통해 귀여운 동물을 키워보세요!

## 🖥️ 주요기능

### 메인화면

- 키우고있는 동물 모습을 볼 수 있다
- 미션, 도감 등 다른 페이지로 이동할 수 있는 아이콘

|               메인화면               |
| :----------------------------------: |
| ![메인화면](/README.assets/main.png) |

### 미션

- 동물에 따라 매일 정해진 미션을 볼 수 있다
- 사진(선택)과 일지를 작성해 미션을 완료할 수 있다
- 메인 미션을 완료한다면 공통 미션을 수행할 수 있다

| 메인미션                                     |      공통미션 |
| :------------------------------------------- | ------------: |
| ![메인미션](/README.assets/main_mission.png) | ![공통미션]() |

### 도감

- 지금까지 다 키운 모든 동물들을 볼 수 있다
- 그 동물을 키우면서 작성한 일지를 볼 수 있다
- 동물에 대한 상세 정보를 볼 수 있다

|                도감                 | 동물 상세                                 |
| :---------------------------------: | ----------------------------------------- |
| ![도감](/README.assets/history.png) | ![동물 상세](/README.assets/history2.png) |

### 커뮤니티 & 공지사항

- 다른 사람들이 작성한 일지를 모두 볼 수 있다
- 적절하지 않은 게시글이 있다면 신고를 통해 성냥을 획득할 수 있다
- 공지사항을 확인할 수 있다

|           커뮤니티           | 공지사항                     |
| :--------------------------: | ---------------------------- |
| ![커뮤니티](/README.assets/) | ![공지사항](/README.assets/) |

### 설정 & 배경화면 설정

- 배경화면 소리 및 효과음 소리의 크기를 설정할 수 있다
- 배경화면을 설정 및 구매할 수 있다

|                설정                 |
| :---------------------------------: |
| ![설정](/README.assets/setting.png) |
